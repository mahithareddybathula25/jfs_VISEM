What is maven ?
Apache Maven is a build automation tool used mainly for java projects
-------------------------------------------------------------------------------------
Why Maven ?
-->without maven
must manually add JAR files
managing versions is different
large projects need complex build scripts
moving project to another system causes dependency issues
-->with maven
automatically downloads all required JARs
maintains proper versioning
simplifies build-->test-->run-->package
standard directory structure
---------------------------------------------------------------------------------------
Key concept ?
-->pom.xml  (project object model)
this file contains the entire project
it has - project name, version, artifactId, dependencies, plugins, build configuration
Ex:-
<project>
	<modelVersion>4.0.0</modelVersion>
	<groupId>com.example</groupId>
	<artifactId>myproject</groupId>
	<version>1.0-SNAPSHOT</version>
	<dependencies>
		<dependency>
			<groupId>mysql</groupId>
			<artifactId>mysql-connector-java</artifactId>		
			<version></version>
		</dependency>
	</dependencies>
</project>
